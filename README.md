# tlhc-crawler
斗六家商的爬蟲跟成績系統
## Demo
[demo](https://tlhc.gnehs.net)
## 啟動
1. 安裝 `npm install`
2. 執行 `node index.js`
3. 搭拉！
## Docker
[Docker Hub](https://hub.docker.com/r/gnehs/tlhc_crawler/)
### 建議掛載檔案/資料夾
`/app/config.js` 首頁的連結和 Slideshow 的設定檔

`/app/ogimage/` 隨機 OGimage 圖片的資料夾
